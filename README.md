# 🧩 Advent of Code

This repository contains my solutions to [Advent of Code challenges](https://adventofcode.com/).

I started this project as a way to work on my algorithmic thinking as a whole — logic, problem-solving, and reasoning — while preparing for upcoming technical interviews.

It’s also a space where I can code more freely and enjoy tackling a variety of problems.

🐍 I’m using Python to stay focused on the algorithms rather than language-specific or framework-related details.

💡 I discovered Advent of Code through a friend, and chose to start with the 2023 edition to progress step by step.

⚙️ I use [gitmoji](https://gitmoji.dev/) to keep my commits clear and consistent, making it easier to quickly understand the purpose of each change.

📝 I try to follow a simple and consistent commit convention throughout the project.

🌱 I don’t use feature branches for this repository. Since Advent of Code is an event with small, self-contained exercises, working directly on the main branch keeps the workflow simple and efficient.
